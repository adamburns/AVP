{{config.APP_NAME}} Documentation

{{config.APP_NAME}} is a tool for advisors to view complex product due diligence documents uploaded by {{config.COMPANY_NAME}} administrators. {{config.APP_NAME}} can ultimately be used to administer any type of document that {{config.COMPANY_NAME}} wants to sell to its user community. Documents are part of Subscription Groups (alternatively, an Admin can set a specific price for a specific document), which may require a paid subscription to access. Documents can also be made available “free of charge” to users of the system.

Registered Users: Dynamic documentation

Log in to view documentation for features and procedures specific to your role.

The home page has some basic information and links to log in or request access.

Request Access

New users must request access to the site using the Request Access form.

Users must enter a valid name, email address and advisor CRD number. If the email address is from a recognized domain, the Broker/Dealer field will auto-populate. Choosing None/Other will display a Broker/Dealer or RIA Name field for manual entry.

Specified admins at {{config.COMPANY_NAME}} will receive an email alert.

Admin Panel

Super Admins, Admins, Home Office Admins and Broker/Dealers have access to the Admin Panel. The Admin Panel is used to manage documents and users. Admin panel features are available within the sidebar on the left, and vary based on the user's Role.

Dashboard

The Dashboard provides an overview of recent users and documents. By default, the 5 most recent of each are displayed. A sum total of all users and documents is displayed at the top of each widget.

You can quickly view or manage users and documents using the buttons that appear upon rolling over each table row. You can also quickly Add or Upload users and documents from the toolbar buttons, or click Manage to go into the full Users/Documents management pages.

Users

You can view, add and remove Users from this page. Admins can manage all users, while Home Office Admins can only edit users within their broker/dealer.

Filter by Broker/Dealer or Role by clicking their names within the table. Search any field using the Search field above the table.

Adding a User

Click the New button to create a new user. Enter the user's email address, first name, last name, broker/dealer, role, and active status.

Active or Inactive?

New users can be active or inactive. Active users are sent an email verification and can log in immediately. Inactive users are 'pre-registered' and must request access before they can log in.

Bulk Uploading Users

Click the Upload button to upload a CSV list of investors. The selected Broker/Dealer and Status will be applied to all users. You can use the template at the top of the page as a sample CSV to get started.

Uploaded files must be CSVs with rows in the following format:

Email,First Name,Last Name,Role

Example: admin@factright.com,John,Smith,admin

Available roles are: superadmin, admin, brokeradmin, broker, advisor, user

Deleting a User

From either the User Info page or the Users table, click the Delete button to delete the selected user. The user is immediately and permanently deleted.

Manage a User

Click the Info button to enter the full User Info page, or Edit to quickly edit the information of a user.

The Info page allows you to reset passwords and view the login, subscription and download history of a given user.

{% if current_user.has_role('superadmin') or current_user.has_role('admin') %}

Admins: Adding/removing Subscriptions

From the Subscriptions tab, you can add and remove free access to selected subscriptions.

Paid subscriptions must be managed through Stripe, easily accessible by clicking the Stripe ID linked in the Profile tab. Cancelled subscriptions will synchronize on the user's next login.

{% endif %}

Documents

All documents potentially available to users are displayed in the Documents table.

Each document's report folder and group are displayed in the Folder column, formatted as Folder (Group). The document's program is displayed in the Programs column. Documents belonging to programs outside of your broker/dealer's valid programs are displayed with a icon.

Filter by Folders or Programs by clicking their names within the table. Search any field using the Search field above the table.

Admins are able to set document prices and advisors can purchase documents from subscription groups.

{% if current_user.has_role('superadmin') or current_user.has_role('admin') %}

Uploading a Document

Click the Upload button to upload a new document. Uploads are restricted to 16MB in size, but can be of any file type.

Select the Report Folder and Program that the document belongs to. For your convenience, the file name field will auto-fill upon choosing a file. You can edit this file name as appropriate.

New Document Notifications

Uploading a document will queue a notification for all users subscribed to the document's subscription group.

By default, a user will receive a single email digest of all of their subscribed uploads every 24 hours, as applicable.

Although not recommended, you can bypass the notification by uploading the document to the 'None' report folder, and reassigning it to a different folder later.

Deleting a Document

From either the Document Info page or the Documents table, click the Delete button to delete the selected document. The document is immediately and permanently deleted.

Manage a Document

Click the Info button to enter the full Document Info page, or Edit to quickly edit the information of a document.

The Info page allows you quickly view the size, report folder, subscription group, program and upload/modification dates for the given document. The Downloads tab displays recent downloads in chronological order.

{% endif %}

Download Report

Admins can view a graph of all downloads by month, as well as filterable tables of all downloads for the selected year.

Valid Programs

Displays the programs your users can view. Contact an admin to adjust your broker/dealer's valid programs.

Data

Admins can edit the Broker/Dealers, Programs, Report Folders, Subscription Groups and Sponsors in the database.

Report Folders belong to a single Subscription Group. Users subscribe to Subscription Groups and can view all Documents from within their Broker/Dealer's Valid Programs. Programs are tracked beneath a single Sponsor.

Broker/Dealers

Users belong to a Broker/Dealer, managed by both their Home Office Admin and all Admins and Super Admins. Broker/Dealers can view documents within selected Valid Programs. Broker/Dealers can have free access to select Subscription Groups.

You can view, add and delete Broker/Dealers from this page.

Adding a Broker/Dealer

Click the New button to add a new broker/dealer. Select any Valid Programs that the Broker/Dealer should begin with.

Deleting a Broker/Dealer

From either the Broker/Dealer Info page or the Broker/Dealers table, click the Delete button to delete the selected broker/dealer. The broker/dealer is immediately and permanently deleted.

Manage a Broker/Dealer

Click the Info button to enter the full Broker/Dealer Info page, or Edit to quickly edit the information of a broker/dealer.

The Info page allows you quickly view users, valid programs, free access groups, creation/modification dates for the given broker/dealer. You can also assign valid Email Domains for the broker/dealer.

Manage Included Access Groups

Admins can grant users within a broker/dealer free access to a subscription group from the Included Access tab.

After granting free access, users must subscribe to the subscription groups as if they were a standard free group. Removing free access from a broker/dealer does not delete subscriptions.

Add Email Domains

The Request Access page can auto-fill the broker/dealer of recognized email addresses of new applicants.

On a Broker/Dealer Edit page, click the New Email Domain button to add a new email domain.

Select the new domain on the Edit Broker/Dealer page. Broker/Dealers can have multiple Email Domains.

Programs

Documents belong to Programs, which Broker/Dealers can flag as Valid Programs. Programs are sponsored by a single Sponsor.

You can view, add and delete Programs from this page. You can filter by Sponsors by clicking a Sponsor within the table.

Adding a Program

Click the New button to add a new program.

Deleting a Program

From either the Program Info page or the Programs table, click the Delete button to delete the selected program. The program is immediately and permanently deleted.

Manage a Program

Click the Info button to enter the full Program Info page, or Edit to quickly edit the information of a program.

The Info page allows you quickly view broker/dealer, documents and creation/modification dates for the given program.

Program Matrix

The Program Matrix visualizes which broker/dealer can view which programs. Broker/Dealers will appear with as Valid for each program they have marked as a Valid Program.

Report Folders

Report Folders are categories that a document can belong to; for example, Closed Opportunities or Open Opportunities. Documents belong to Report Folders, and Report Folders belong to Subscription Groups.

You can view, add and delete Report Folders from this page. You can filter by Groups by clicking a Group in the table.

Adding a Report Folder

Click the New button to add a new report folder. Select the Subscription Group that the Folder will belong to.

Deleting a Report Folder

From either the Report Folder Info page or the Report Folders table, click the Delete button to delete the selected report folder. The report folder is immediately and permanently deleted.

Manage a Report Folder

Click the Info button to enter the full Report Folder Info page, or Edit to quickly edit the information of a report folder.

The Info page allows you quickly view creation/modification dates for the given report folder.

Subscription Groups

Subscription Groups contain one or multiple Report Folders. Users subscribe to Subscription Groups to view all documents within the Group's Folders.

You can view, add and delete Subscription Groups from this page.

Paid Groups

Setting a price for a group will permanently lock in that price. You must create a new group if you wish to adjust pricing.

This rule is enforced by Stripe to prevent the price of a subscription changing after a user has already subscribed.

Adding a Subscription Group

Click the New button to add a new subscription group. Add a price to create a Paid Group with a corresponding Stripe plan.

Deleting a Subscription Group

From either the Subscription Group Info page or the Subscription Groups table, click the Delete button to delete the selected subscription group. The subscription group is immediately and permanently deleted. The Stripe plan of a Paid Group will remain in Stripe's system.

Manage a Subscription Group

Click the Info button to enter the full Subscription Group Info page, or Edit to quickly edit the information of a subscription group.

The Info page allows you quickly view creation/modification dates for the given subscription group.

You can add a price to a previously-free group from the Edit page; be forewarned that the group will be permanently locked in at this price.

Sponsors

Programs belong to a Sponsor for tracking and filtering purposes.

You can view, add and delete Sponsors from this page.

Adding a Sponsor

Click the New button to add a new sponsor.

Deleting a Sponsor

From either the Sponsor Info page or the Sponsors table, click the Delete button to delete the selected sponsor. The sponsor is immediately and permanently deleted.

Manage a Sponsor

Click the Info button to enter the full Sponsor Info page, or Edit to quickly edit the information of a sponsor.

The Info page allows you quickly view creation/modification dates for the given sponsor.

User Panel

Users can manage their subscriptions, notifications and view documents from within their personal User Panel.

Dashboard

The User Dashboard shows newly uploaded documents and a shortcut to the Subscription page.

The 5 most recent documents are displayed in a small table. The total of new documents since the last login is shown in the dashboard header.

Documents

Users can view and download documents from subscribed groups on this page.

New documents since the last login are flagged with New.

Subscriptions

Users can subscribe to subscription groups from this page.

Notifications

Users receive an email digest of recently uploaded documents from within their subscriptions.

You can disable notifications for a given group by clicking the notification status icon in your Subscriptions table.

Included Groups

All users can subscribe to included groups simply by clicking 'Subscribe'. New users are subscribed to all included groups by default.

Paid Groups

Users can pay for groups by clicking 'Purchase' and following the Stripe checkout process. Users must enter a valid credit or debit card to purchase the group.

Subscriptions last for one year from the date of purchase. By default, the subscription will rebill automatically. Users can toggle auto-billing by clicking the Subscription status link.

Subscription emails are managed by Stripe. Stripe sends billing notifications within a few weeks of the resubscription date.

Manage Account

Users and admins can change their personal information and email address on this page.

Email Address Changes

Changing your email address will lock your account until you confirm the new address. Please make sure you use a valid email address.

Email Alerts

Select how often you would like to receive subscription notifications. {{config.APP_NAME}} will create an email digest for all documents within the selected frequency.

Selecting 'Immediately' will send email alerts during the report mail-out every day.

Time Zone

Select your preferred time zone. All timestamps on {{config.APP_NAME}} will reflect your choice in time zone.
